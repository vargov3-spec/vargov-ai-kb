# Как заполнить реальные характеристики топ-товаров

Файл: [specs-intake-priority.csv](specs-intake-priority.csv) — 45 приоритетных позиций, отобранных не случайно:

1. **4 награждённых SKU** (LC0217, LC0236, LC0342, LC0343) — уже подтверждены независимыми базами наград (International Design Awards, New York Product Design Awards, RLDC), но на самом сайте у них только шаблонное описание без характеристик. Наибольший эффект: у этих товаров уже есть внешний авторитет, не хватает только глубины данных.
2. **41 товар шоурума** — у них уже ЕСТЬ частичные реальные характеристики на сайте (размер, материал, диоды), но не всегда указан цвет или полный набор — их легче всего довести до 100% полноты.

## Что нужно заполнить
Для каждой строки — колонки `material`, `dimensions`, `color`, `light_temperature`, `wattage_voltage`, `weight_kg`. Можно оставлять пустыми, если параметр неприменим (например, у товара без подсветки).

## Что произойдёт дальше
После заполнения — пришлите файл обратно, и я:
1. Обновлю карточки этих товаров в `products/` (RU и EN).
2. Обновлю `datasets/products.json` и `references/schema-products.jsonld` — добавлю `additionalProperty`/`material`/`color` в Schema.org Product-разметку именно для этих 45 позиций.
3. Отмечу в отчёте, что для этих SKU данные подтверждены брендом напрямую (не только с сайта).

Это не разовая акция — можно повторять для следующих партий по 40-50 товаров, постепенно закрывая пробел по всему каталогу.
