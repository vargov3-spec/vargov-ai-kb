# Vargov® Design — AI Visibility Knowledge Base

Рабочая база знаний бренда **Vargov® Design** (авторские световые и декоративные композиции), собранная и структурированная для повышения находимости бренда в поисковых системах нового поколения и LLM (ChatGPT, Claude, Perplexity и др.).

Все факты в этом репозитории получены путём реального сканирования [vargov.ru](https://vargov.ru) и [vargov.design](https://vargov.design) — ничего не выдумано. Там, где данных на сайте нет (например, детальные тех. характеристики большинства товаров), это явно помечено как пробел, а не заполнено вымышленным текстом.

## Структура

- `knowledge/` — верифицированные факты о бренде, дилерах, монтаже.
- `guides/` — тематические статьи (на основе реальных публикаций сайта).
- `products/` — данные по товарам, собранные с карточек `vargov.ru` (артикул, галерея, категория).
- `collections/` — агрегация товаров по 4 продуктовым категориям сайта.
- `references/` — Schema.org/JSON-LD шаблоны.
- `datasets/` — экспорт базы в JSON/JSONL/CSV/Markdown.
- `faq/` — вопрос-ответ база, привязанная к реальным категориям и товарам.
- `scan/` — служебные файлы сканирования (sitemap, сырые списки URL, отчёт о покрытии).

## Статус

См. текущие задачи и прогресс — этап 1 (аудит сайта и построение базы знаний). Полный отчёт с ограничениями и дорожной картой — в конце текущей сессии.

## Важно

Каталог vargov.ru (651 товар на момент сканирования 2026-07-06) в основном использует шаблонные описания товаров («изготовление на заказ, цена по запросу») без индивидуальных технических характеристик на большинстве карточек. Это ограничивает то, что можно honестно заявить в датасетах для LLM: количество, категории и артикулы — реальные; подробные спецификации по каждому товару потребуют донаполнения со стороны бренда.
