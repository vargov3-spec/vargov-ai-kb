# Vargov® Design — Installation and Connection

Source: [vargov.ru/installation_and_connection_vargovdesign_ru](https://vargov.ru/installation_and_connection_vargovdesign_ru) (verified 2026-07-06).

## Receiving the shipment
- Check the packaging for damage on arrival and photograph any defects.
- The 1:1 scale installation diagram is printed on banner fabric and packed inside one of the boxes.

## Surface preparation (drywall ceilings)
- Install plywood backing plates (9mm minimum thickness) at mounting points.
- Drill holes 8–14mm in diameter following the 1:1 installation diagram.

## Electrical connection — 12V (LED) systems
- Reserve space for the driver in the electrical panel or ceiling cavity.
- Run low-voltage wiring from the driver to the composition's location (one line per element).
- Route wires through the prepared holes.
- Connect using the included terminal blocks, observing polarity.

## Electrical connection — 220V systems
- Run 220V cable to the installation location.
- Split the wiring per the number of elements.
- Route wires through the prepared holes.
- Connect wires to the elements using the included terminal blocks.

## Wire labeling
- Each wire is tagged with a number matching the installation diagram.
- The tags also act as height limiters, preventing installation errors and ensuring correct element alignment.

## Safety
- Disconnect the power supply before starting any work.
- Use quality terminal blocks and heat-shrink tubing.
- Follow all applicable technical standards.

## Recommendation
Given the complexity of installation, official Vargov®Design installers are strongly recommended (this service is available in Russia and the UAE).
